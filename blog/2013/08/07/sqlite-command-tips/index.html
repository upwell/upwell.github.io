SQLite Command Tips

Aug 7th, 2013

Use a database file

sqlite <database_file>

Get table information

.tables

Get schema information

.schema
.schema table_name

Get help

.help

Write result to a file

.mode list
.separator |
.output result.data
select * from table_name;
.exit

Exit

.exit
